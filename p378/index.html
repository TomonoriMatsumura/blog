さくらのスタンダードでsftpを使う

2012-12-07
Share
  Tweet  

EclipseのRSE(Remote System Explore)を使おうと思い、さくらのスタンダードにEclipseから一向に接続できない。どうやら原因はこれらしい。。。

.cshrcの設定を変更する

~/.cshrcを開いて以下を書き加える

if ( $?SSH_CLIENT &amp;&amp; ! $?SSH_TTY ) then
setenv LANG C
unsetenv LC_ALL

else if ( $?prompt ) then

接続テスト

sftp -v user@domain
