Ansibleでデバグを行う際に環境変数を調べる必要が出てきた。

### 環境変数を取得する

コマンド

```:bash
ansible -m setup hostname

# VPSに最初に接続する例
# インベントリーファイルを指定してparamikoを利用する
export ANSIBLE_HOST_KEY_CHECKING=False; ansible -u root -k -i /path/to/inventory_file -m setup "160.xx.xxx.xxx," --ask-vault-pass -c paramiko &gt; setup_envs
```
