### VLC media playerのインストール準備

```:bash
yum install epel-release
yum localinstall --nogpgcheck http://li.nux.ro/download/nux/dextop/el7/x86_64/nux-dextop-release-0-1.el7.nux.noarch.rpm

yum clean all
yum update
```

### VLC media playerをインストールする

```:bash
yum install vlc
```
