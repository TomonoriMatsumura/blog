### 圧縮する

```:bash
# tar (指定したファイル郡を一つのアーカイブファイルにまとめる)
tar cfv files.tar files/

# tar + gzip
tar cfvz files.tar.gz files/
```

### 解凍する

```:bash
# tar (tarアーカイブを展開)
tar xfv files.tar

# tar + gzip
tar xfvz files.tar.gz 
```
